# Portfolio Interactivo Personal 🎨

Este proyecto es el resultado final del **Módulo 6: Fundamentos del Diseño Visual y Desarrollo Web**. Se trata de una landing page profesional diseñada para comunicar mi identidad visual y mostrar mis habilidades y proyectos.

## 📋 Objetivos Cumplidos
- [x] **Diseño Visual:** Aplicación de principios de alineación, contraste y jerarquía.
- [x] **Semántica:** Estructura HTML5 clara y accesible.
- [x] **Responsividad:** Adaptable a móviles, tablets y escritorio.
- [x] **Accesibilidad:** Validación de contraste según WCAG 2.1.

## 🎨 Decisiones de Diseño
* **Tipografía:** [Cormorant Garamond y DM Sans] seleccionadas por su legibilidad y carácter profesional.
* **Paleta de Colores:** Basada en tonos [cálidos/neutros] para transmitir [cercanía y elegancia].
* **Estructura:** Diseño *mobile-first* priorizando la navegación táctil.

## 🛠️ Tecnologías Usadas
* HTML5 (Semántico)
* CSS3 (Flexbox/Grid y Media Queries)
* Animaciones CSS (fadeUp)
